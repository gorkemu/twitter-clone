## Installation  

1. Clone the repository:  
```bash  
git clone https://github.com/your-username/twitter-clone.git  
```  

2. Install dependencies:  
```bash  
npm install  
```  

3. Firebase configuration:  
- Update `src/firebase.js` with your Firebase settings  

4. Start the app:  
```bash  
npm start  
```
